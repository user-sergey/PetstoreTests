**Проект содержит автотесты для сервиса petstore (https://petstore.swagger.io/).**

Стек: Kotlin, JUnit.

**Запуск тестов:**
1. Открыть командную строку и перейти к проекту:

`cd .../PetstoreTests`

2. Прописать команду

`gradlew.bat test`

**ИЛИ**

1. Открыть проект в IntelliJ IDEA.
2. Открыть терминал (Alt+F12 или View - Tool Windows - Terminal) и прописать:

`./gradlew test`

---

Файл со всеми тестами расположен здесь: `src/test/kotlin/ApiTests.kt`
